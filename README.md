# Rock Paper Scissors Game

A simple console-based Rock Paper Scissors game implemented in C++ (C++14). The game supports two modes: Player vs Computer and Player vs Player.

## Features

- **Player vs Computer:** Play against a computer opponent with random choices.
- **Player vs Player:** Two players can play against each other on the same machine.
- **Clear instructions and menu navigation.**
- **Quit anytime:** Enter `q` to exit a game mode.

## How to Play

1. **Run the program.**
2. **Choose a mode:**
   - `1` for Player vs Computer
   - `2` for Player vs Player
   - `3` to Exit
3. **Make your move:**
   - Enter `r` for Rock
   - Enter `p` for Paper
   - Enter `s` for Scissors
   - Enter `q` to quit the current mode

## Build Instructions

1. Open the project in Visual Studio 2022.
2. Build the solution (__Build Solution__).
3. Run the executable (__Start Without Debugging__).

## File Structure

- `main.cpp` — Entry point and game loop logic.
- `RPSGame.h` / `RPSGame.cpp` — Game class with helper methods for menu, computer choice, and displaying choices.
